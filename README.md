# Interactive 3D Product Viewer

An interactive 3D web application built with React and Three.js that allows users to explore a 3D object in real time.

## 🚀 Features

- Rotate 3D object (mouse drag)
- Zoom in/out (scroll)
- Real-time rendering using WebGL
- Clean and responsive UI
- Built with modern frontend tools

## 🛠️ Tech Stack

- React (Vite)
- Three.js
- React Three Fiber
- Drei

## 📦 Installation

```bash
npm install
npm run dev
```
